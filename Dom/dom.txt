
structure of DOM
Document
└── DocumentType (<!DOCTYPE html>)
└── Element (html)
    └── Element (head)
        └── Element (meta)
        └── Element (meta)
        └── Element (title)
            └── Text node ("DOM Basics")
    └── Element (body)
        └── Element (header)
            └── Element (h1)
                └── Text node ("Welcome to DOM Basics")
        └── Element (main)
            └── Element (p)
                └── Text node ("This is a simple example of the DOM structure.")
            └── Element (ul)
                └── Element (li)
                    └── Text node ("Item 1")
                └── Element (li)
                    └── Text node ("Item 2")
                └── Element (li)
                    └── Text node ("Item 3")
        └── Element (footer)
            └── Element (p)
                └── Text node ("&copy; 2024 DOM Basics")
